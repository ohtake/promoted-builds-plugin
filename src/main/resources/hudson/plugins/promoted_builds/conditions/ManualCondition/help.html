Designates that this promotion as requiring someone to go to the Hudson web UI and click the "approve promotion" button. This is useful when the promotion process is controlled by humans (such as by voting), or when you'd like to temporarily suspend promotion (such as when you are nearing a release.)

You can also optionally list user names or groups that are allowed to approve the promotion. This list should be comma delimited.

If you add '!' char in front of user names or groups, those marked are not allowed to approve. For instance, "!anonymous" allows Non-anonymous to approve, and "group1,!user1" allows members in group1 except user1 to approve.
